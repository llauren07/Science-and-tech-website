Département d'architecture : construction d'une pyramide

Corrigé de l'exercice "Département d'architecture : construction d'une pyramide":


nbMaxPierres = int(input())

cote = 0
nbPierresActu = 0

while (nbPierresActu <= nbMaxPierres):
    cote += 1
    nbPierresActu += cote * cote

nbPierresActu -= cote * cote
cote -= 1

print(cote)
print(nbPierresActu)

Département de chimie : mélange explosif

Corrigé de l'exercice "Département de chimie : mélange explosif":


nbTotalMesures = int(input())
tempMin = int(input())
tempMax = int(input())

doitContinuer = True

count = 0

while (doitContinuer):
    tempActu = int(input())
    count += 1

    if (tempActu >= tempMin and tempActu <= tempMax):
        print("Rien à signaler")
    else:
        print("Alerte !!")
        doitContinuer = False

    if (count >= nbTotalMesures):
        doitContinuer = False
