Grand inventaire

Corrigé de l'exercice "Grand inventaire":


stock=[0]*10
nbOperations =int(input())
for i in range(nbOperations):
    ingredient=int(input())
    quantite=int(input())
    stock[ingredient-1]+=quantite
for i in stock:
    print(i)

Répartition du poids

Corrigé de l'exercice "Répartition du poids":


nbCharrettes =int(input())
masseCharrettes=[0.0]*nbCharrettes
somme=0
for i in range(nbCharrettes):
    masse=float(input())
    masseCharrettes[i]+=masse
    somme+=masse
moyenne=somme/nbCharrettes
for i in masseCharrettes:
    print(moyenne-i)

Visite de la mine

Corrigé de l'exercice "Visite de la mine":


nbDeplacements=int(input())
deplacements=[0]*nbDeplacements
for i in range(nbDeplacements):
    deplacement=int(input())
    deplacements[i]+=deplacement
deplacements.reverse()
for i in deplacements:
    if i==5:
        print(4)
    elif i==4:
        print(5)
    elif i==1:
        print(2)
    elif i==2:
        print(1)
    else:
        print(3)

Journée des cadeaux

Corrigé de l'exercice "Journée des cadeaux":


nbHabitants = int(input())

fortunes = [0] * nbHabitants

for i in range(nbHabitants):
    fortunes[i] = int(input())
    
fortunes.sort()

fortuneMediane = 0
indiceMoyen = (nbHabitants - 1) // 2
if (nbHabitants % 2 == 1):
    fortuneMediane = fortunes[indiceMoyen ]   
else :
    fortuneMediane = (fortunes[indiceMoyen ] + fortunes[indiceMoyen + 1]) / 2
    
print(fortuneMediane)

Course à trois jambes

Corrigé de l'exercice "Course à trois jambes":



nbParticipants = int(input())

nombres = [0] * nbParticipants

for i in range(nbParticipants):
    nombres[i] = int(input())
    
nombres.sort()

for i in range(nbParticipants // 2):
    print("{} {}".format(nombres[i], nombres[nbParticipants - 1 - i]))
